# Prompts usados para setup inicial de CineGlobal

Este archivo recopila todos los prompts pedidos al asistente para configurar el proyecto.

1. En la rama activa creame los siguientes archivos y carpetas:
   docs/
   docs/01-mockup/
   docs/02-prompts/
   docs/03-specs/
   index.html
   README.md
   plan.md
   changelog.md

2. hace un commit con el mensaje: feat(setup): Initial project structure. Luego push de la rama feature/devops-project-setup

3. Generá en el archivo plan.md en la raíz del repositorio para el proyecto CineGlobal lo siguiente: ... (especificación maestra con objetivo, alcance, actores, requerimientos funcionales, reglas/suposiciones, funcionalidades futuras)

4. Correji todos las palabras con tilde porque no se visualiza la tilde. Guarda el archivo en UTF-8

5. Cambia el contenido de plan.md. Es demasiado ambicioso. Esto es para una materia de programacion web donde estamos empezando a ver front end y no vamos a ver backend.

6. No menciones que es para una materia y no es para movil. Es solo web.

7. Generá el archivo docs/03-specs/spec-devops.md para el proyecto CineGlobal. ... (especificación de rol DevOps con objetivos, alcance, flujo de trabajo, criterios de aceptación)

8. Genera el archivo docs/02-prompts/prompt-spec-devops.md. En este archivo van a ir todos los prompts que utilice para el setup inicial del proyecto. Agrega todos los prompts que utilice.
