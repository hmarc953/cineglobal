# Spec Frontend - Proyecto CineGlobal

## Objetivo

Desarrollar una página web utilizando HTML5 para mostrar la cartelera de películas disponibles en distintos cines, incluyendo filtros por ubicación y categoría.

## Alcance

La página incluirá:

- Header con título del sitio
- Navegación con filtros por cine y categoría
- Sección principal con cartelera de películas
- Fecha
- Horarios
- Enlaces de compra de entradas
- Footer informativo

## Justificación

Se utiliza HTML5 para mejorar:
- La accesibilidad del sitio, su estructura, para mantener el codigo prolijo y legible. Y para poder mantener el posicionamiento SEO. 


## Estructura del documento

Se utilizarán las siguientes etiquetas:

- header
- nav
- main
- section
- article
- footer
- time
- data

## Criterios de aceptación

El desarrollo será considerado correcto si:

- El HTML valida sin errores
- Se utilizan etiquetas semánticas correctamente
- Cada película está dentro de un article
- Existe navegación con filtros
- Se incluyen fechas con data
- Se incluyen horarios con time
